Import Subscribers

Instructions
  1. Upload an Excel spreadsheet (.xlsx) containing data in the template file that was downloaded. (Download Template>)
    Only columns that are specified in the provided template will be uploaded. The supported columns are CODE, NAME, PHONE_NUMBER, GENDER, DATE_OF_BIRTH, LOCATION, START_DATE, VOICE, SMS, LAND_SIZE, LANGUAGE, DISTRICT, EDUCATIONAL_LEVEL, SUBSCRIBER_TYPE, SUBSCRIBER_GROUP_ (1 - 5), SUBSCRIBER_PRIMARY_COMMODITY, SUBSCRIBER_COMMODITY_ (2 - 5) and PROGRAM
    For some columns, you will have to select an option from the options list that will be provided for those columns.
  2. To upload a file, click on the browse to slect the filled template file and then click on the Upload button to upload the data.
  3. If there are errors in the file, the system will prompt you and the upload file will be rejected.
